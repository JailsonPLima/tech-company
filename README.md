# Como executar o projeto

Pré-requisitos: `npm`

Primeiramente, clone este projeto com o código abaixo.
``` bash
git clone git@github.com:JailsonPLima/tech-company.git
```

Então entre na pasta do projeto que acabou de clonar e execute: 
``` bash
npm install
```

E para inicializar:
``` bash
npm run dev
```
